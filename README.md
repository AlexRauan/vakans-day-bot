
# Vakans Day Bot

Телеграм-бот для флирта, работает 24/7 на Render.

## Развёртывание

1. Установи зависимости:

```
pip install -r requirements.txt
```

2. Запусти локально:

```
BOT_TOKEN=твой_токен python bot.py
```

3. Для Render:

- Укажи в **Environment → Secret**: `BOT_TOKEN=твой_токен`
- Деплой через GitHub
- Profit!
